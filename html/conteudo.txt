Conteúdos estáticos:
- São os conteudos que não realizam alterações, e são fixos

Conteúdos dinamicos:
- Um código que gera o html, e envia para o browser, realizando alterações na página sempre que for necessário

Servidor DNS:
- Traduz os nomes de domínio em um endereço de IP

HTTPSeguro
- Dados criptografados usando protocolos TLS ou SSL
- Protege contra interceptação
- usa certificados digitais

WEB SERVICE
- Serviços da WEB
- É um serviço que roda atraves de requisições da WEB
- O postman serve para testar os web services que criamos, fazendo diversas requisições
- POST - Novo cliente
- GET - Obter todos os clientes
- PUT - Atualizar o cliente 
- DELETE - Excluir o cliente
